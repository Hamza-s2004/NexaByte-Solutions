### Description
This project demonstrates how unauthorized webcam access can occur 
through a browser. Built for educational and ethical hacking purposes only.

### Features
- Auto webcam activation on page load
- Video saved to Local Storage (Phase 1)
- Video uploaded to Dropbox (Phase 2)
- Device information capture (Bonus)
- Security awareness warning popup

### Tech Used
- HTML, CSS, JavaScript
- MediaRecorder API
- Dropbox API
- GitHub Pages (Deployment)

### Disclaimer
This project is for educational purposes only. 
Unauthorized camera access is illegal and a violation of privacy.
